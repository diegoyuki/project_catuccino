# Project_Catuccino
Proyecto final para TripleTen de FrontEnd basado en un sitio web de un Cat Cafe ficticio tomando gran inspiracion en la pagina Cat Cafe Mocha.

**Catuccino** es una plataforma web moderna y profesional diseñada para un café de gatos ubicado en Monterrey, México. El sitio permite a los usuarios conocer a los residentes felinos, explorar el menú, gestionar reservaciones y localizar la sucursal física.

## Características

*   **Interfaz Ultra-Responsiva:** Diseño optimizado para una visualización perfecta en dispositivos desde pantallas grandes hasta teléfonos de 256px.
*   **Gestión de Reservaciones:** Sistema integrado para agendar visitas al café.
*   **Catálogo de Residentes:** Sección dedicada para conocer a los gatos como Moka, Pantera, Copito y más.
*   **Navegación Inteligente:** Barra de navegación dinámica que adapta su contraste (modo oscuro) según la sección.
*   **Experiencia Inmersiva:** Uso de videos en segundo plano y efectos Parallax para una navegación moderna.

## Tecnologías Utilizadas

*   **Frontend:** React.js
*   **Enrutamiento:** React Router DOM
*   **Estilos:** CSS3 con arquitectura BEM y animaciones personalizadas
*   **Herramientas:** GitHub Copilot y servicios de Google Cloud

## Estructura del Proyecto

*   `src/components/`: Componentes modulares como `Header`, `Main`, `Reservaciones` y `Location`.
*   `src/images/`: Recursos visuales del sitio.
*   `src/utils/`: Lógica de integración con APIs y utilidades de datos.

## Instalación y Uso

1.  **Clona el repositorio:**
    ```bash
    git clone [https://github.com/diegoyuki/catuccino.git](https://github.com/diegoyuki/catuccino.git)
    ```
2.  **Instala las dependencias:**
    ```bash
    npm install
    ```
3.  **Inicia el servidor de desarrollo:**
    ```bash
    npm start
    ```
